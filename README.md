## Tech Stack

## Require
* npm || npm3
  * installation
  <pre>sudo apt install npm</pre>
* bower
  * installation
  <pre>(sudo) npm install bower</pre>

## Executiton
* node_modules dependency
<pre>$ npm install</pre>
* bower_components dependency
<pre>$ bower install</pre>
* .env modify (DB_HOST, PORT, etc...)
<pre>$ cp .env.sample .env</pre>
* server execution
<pre>$ node server</pre>
